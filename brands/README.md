# Brand assets

Logo and icon for the **Growatt Modbus** integration: the green Growatt **G**
(`#8CBD3A`) paired with "modbus" in the Growatt wordmark grey (`#606060`).

| File | Size | Use |
|------|------|-----|
| `icon.png` | 256×256 | square app icon (G over "modbus") |
| `icon@2x.png` | 512×512 | hi-DPI icon |
| `logo.png` | 1078×256 | wide logo (G + "modbus") |
| `logo@2x.png` | 2158×512 | hi-DPI logo |

These mirror the integration's entry in
[home-assistant/brands](https://github.com/home-assistant/brands/tree/master/custom_integrations/growatt_modbus),
which is what Home Assistant and HACS actually display. Update both together.
